# Dotfiles

Personal dotfiles.

## Installation

### Bootstrap

To get the repo and bootstrap any requirements, you'll want to do a git clone and run the included bootstrap shell script.

```
$ git clone git@github.com:eugenetriguba/dotfiles.git
$ cd dotfiles
$ chmod u+x bootstrap
$ ./bootstrap
```

### Run

```
$ chmod u+x run
$ ./run
```
